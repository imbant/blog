个人博客，记录所思所想。

- 基于 [Hexo](https://hexo.io/) 构建的静态网站。
- 使用 GitHub Action 持续集成服务。
- 部署在 GitHub Pages 上。

如何启动：

```bash
node -v # 13.14.0，与 node-sass 等相关

npm i

npm run serve
```
